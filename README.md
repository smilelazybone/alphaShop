### install

1.打開 terminal，Clone 此專案至本機電腦

```
git clone https://github.com/smilelazybone/alphaShop.git
```

2.開啟 terminal

```
cd my-app
```

3.安裝 npm 套件

```
npm install
```

4.開啟專案

```
npm start
```

5.出現以下內容表示伺服器已啟動

```
Local:   http://localhost:3000/
